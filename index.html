Inspetor de Elementos

Voltar

O inspetor de elementos é uma ferramenta utilizada para obter informações de uma página HTML, permitindo até mesmo a alteração do seu conteúdo e a observação da construção da página em questão.

Estrutura Básica do HTML

Voltar

O HTML funciona por meio de tags. Para iniciar um projeto HTML, utilizamos a tag < html > , indicando ao navegador que o tipo de arquivo trabalhado é HTML. A estrutura interna básica possui duas tags, < head > e < body >.

1.Tag head

A tag < head > é responsável pela parte não visível para o usuário na página HTML. Essa seção contém informações que são carregadas antes do usuário entrar na página (no lado do cliente). Um exemplo de conteúdo comumente encontrados dentro da tag < head > é a tag < title > que define o título da página, que é exibido na barra de título do navegador.

2.Tag body

A tag < body > é responsável pelo conteúdo que é visível para o usuário na página HTML. Esta seção contém informações que o usuário pode facilmente encontrar e interagir. Alguns exemplos de conteúdo comumente encontrados dentro da tag < body > incluem:

  • Texto: Parágrafos, cabeçalhos, listas e outros elementos de texto que compõem o conteúdo da página.
  • Imagens: Elementos visuais que são exibidos na página, utilizando a tag < img >.
  • Links: Hiperlinks criados com a tag < a > que permitem a navegação para outras páginas ou recursos.< /p >

Tags

Voltar

As tags são elementos fundamentais que são usados para estruturar o conteúdo em uma página da web. As tags são cercadas por colchetes (<>) e geralmente ocorrem em pares de abertura e fechamento, embora algumas tags não exijam uma tag de fechamento. Algumas das principais tags:

  • Strong: A tag strong é utilizada para destacar alguma parte do texto
    EX:
    Essa parte é Importante
  • Input: A tag input é uma tag HTML usada para criar campos interativos em formulários web. Ela é um elemento que não possui uma tag de fechamento. Pode ter diversos tipos, dependendo do tipo de entrada desejado. Alguns dos atributos mais comuns incluem:
    • type: Define o tipo de campo de entrada. Pode ser "text" para caixas de texto, "checkbox" para caixas de seleção, "radio" para botões de rádio, "submit" para botões de envio, entre outros.
    • name: Especifica o nome do campo, que será usado para identificar o valor do campo quando o formulário for enviado.

  • Img: A tag < img > em HTML é usada para incorporar imagens em uma página web. Aqui estão os principais atributos associados a essa tag:
    • src (source): Este atributo especifica o caminho (URL) da imagem. Pode ser um caminho relativo ou absoluto para a imagem.
    • alt (alternative text): Este atributo fornece um texto alternativo para a imagem, que é exibido se a imagem não puder ser carregada. Além disso, é utilizado por leitores de tela para acessibilidade.
    • width e height: Esses atributos especificam a largura e a altura da imagem, respectivamente. Eles podem ser definidos em pixels ou porcentagens.
    • title: Este atributo fornece um título para a imagem. O título é frequentemente exibido quando o usuário passa o mouse sobre a imagem.

Atributos das Tags

Os atributos HTML são informações adicionais fornecidas em tags HTML para modificar ou configurar o comportamento de um elemento. Eles são sempre incluídos dentro da tag de abertura do elemento e têm a forma "nome="valor"". Alguns exemplos de atributos:

  • id: Define um identificador exclusivo para o elemento.
  • class: Define uma ou mais classes que podem ser usadas para aplicar estilos ou selecionar elementos com CSS ou JavaScript.
  • src: Define o caminho (URL) da imagem.
  • alt: Fornece um texto alternativo para a imagem, útil para acessibilidade.

Listas

Voltar

As listas podem ser classificadas em dois tipos principais: listas ordenadas, representadas pela tag < ol >, e listas não ordenadas, representadas pela tag < ul> . Em ambas as categorias, é essencial incluir os itens individuais utilizando a tag < li > para indicar cada elemento da lista.
Ex:
Lista não ordenada:

  • Item 1
  • Item 2
  • Item 3

Lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Links

Voltar

links são elementos que permitem a navegação entre diferentes páginas da web ou recursos online. A tag usada para criar links é < a >, e ela pode ser configurada com vários atributos.

< a href="https://www.exemplo.com">Exemplo.com

  • href: Especifica o URL de destino para o link. Pode ser uma URL absoluta, relativa ou um link para um elemento no mesmo documento.
  • target: Define onde o link será aberto. Por exemplo, _blank abre o link em uma nova janela ou guia.
  • rel: Indica a relação entre a página atual e a página vinculada. O valor "nofollow" é usado para dizer aos motores de busca que não sigam o link.
  • title: Fornece um texto de dica que é exibido quando o usuário passa o mouse sobre o link.
